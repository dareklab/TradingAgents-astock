# A股权威媒体信息披露平台

---

## 官方信息披露平台

| 平台 | 说明 | 链接 |
|------|------|------|
| **巨潮资讯网** | 证监会指定信息披露网站 — 上市公司原始公告PDF | [cninfo.com.cn](https://www.cninfo.com.cn) |
| **上海证券交易所** | 沪市主板 + 科创板公告、监管信息 | [sse.com.cn](https://www.sse.com.cn) |
| **深圳证券交易所** | 深市主板 + 创业板公告、监管函件 | [szse.cn](https://www.szse.cn) |
| **北京证券交易所** | 北交所公告、信息披露 | [bse.cn](https://www.bse.cn) |
| **中国证监会** | 政策法规、行政处罚、IPO注册信息 | [csrc.gov.cn](https://www.csrc.gov.cn) |

## 使用指南

### 查询上市公司原始公告

- **巨潮资讯网** → 首页搜索框输入股票代码 → 查看全部公告PDF原文
- **上交所** → 「信息披露」→「上市公司公告」→ 按代码或日期检索
- **深交所** → 「信息披露」→「上市公司信息」→ 公告查询

### 获取权威新闻

- **中国证券报** ([cs.com.cn](https://www.cs.com.cn)) — 证监会指定信息披露报纸
- **上海证券报** ([news.cnstock.com](https://news.cnstock.com)) — 新华社主办
- **证券时报** ([stcn.com](https://www.stcn.com)) — 人民日报社主办
- **证券日报** ([zqrb.cn](https://www.zqrb.cn)) — 经济日报社主办

> **提醒**：以上数据均来源于公开的官方信息披露平台。巨潮资讯网（cninfo.com.cn）为证监会指定唯一法定披露网站，上市公司公告以巨潮发布的PDF原文为准。
